Privacy Notice

This form uses VeilForms client-side encryption. Your data is encrypted in your browser before being transmitted. We receive only encrypted blobs - we cannot read your message without the decryption key.

Testing Mode

To test what Netlify receives:

  1. Submit the form above
  2. Go to your Netlify Dashboard
  3. Navigate to: Site > Forms > contact
  4. View the submission - you'll see encrypted blobs instead of readable data

Note: Currently running without a public key, so data is sent unencrypted for initial testing. Add a public key to enable encryption.
